# Hayat Asistanı (AI-Assisted Cross-Platform Mobile Application)

An ongoing, highly modular cross-platform mobile assistant application architected and developed in tight collaboration with an **AI Coding Partner**. The project is strategically built utilizing the Flutter framework and Dart language to streamline daily life tracking, including workouts, financial logging, dietary tracking, and mindfulness routines.

## 🤖 AI-Human Collaboration Showcase
This repository serves as a direct portfolio piece demonstrating efficient prompt engineering, dynamic architectural planning, and pair-programming with AI. 
- **AI Role:** Assisted in structural folder mapping, component modularization code patterns, and dynamic view routing logic.
- **Developer Role (Kemal):** Engineered core layouts, custom widget overrides, state operations, file management, and production pipeline assemblies.

---

## 🚀 Key App Domains & Modules
The app features an integrated multi-screen dashboard designed to act as a unified personal companion:
- **`finans_screen.dart` & `diyet_screen.dart`:** Modular input sheets to track user balances, expenses, and caloric budgets.
- **`antrenman_screen.dart` & `meditasyon_screen.dart`:** Time-tracked logging dashboards built for physical and mental well-being tracking.
- **`kitap_screen.dart` & `dizi_film_screen.dart`:** Personalized culture catalog trackers utilizing custom model entities (`kitap_model.dart`).

---

## 📁 Repository Directory Structure
- `lib/models/`: Encapsulates decoupled data structures (`user_model.dart`, `kitap_model.dart`) mapping clean parameters for systemic integrity.
- `lib/screens/`: Independent layout modules separating structural navigation layers and UI widget definitions.
- `pubspec.yaml`: Governs external dependencies, responsive assets configurations, and system SDK environments.
